How it works

An Elo rating predicts each game; a nine-stage read of each team's form is layered on top. The engine "calls" a game only when the favorite is in a breakout stage (8-9) AND confidence is at least 62% — the A+ slice, roughly 5% of games, which hit about 70% in historical and cross-source holdout testing. On every other game it stays silent. Live track record builds over the season.
